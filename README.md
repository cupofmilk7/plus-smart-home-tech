# Smart Home Tech

Проект Smart Home Tech представляет собой комплексную систему для управления умным домом. Проект включает в себя несколько модулей, охватывающих различные аспекты систем управления умным домом, включая сбор данных с датчиков, анализ данных, управление заказами, платежи и доставку.

## Структура проекта

Проект состоит из следующих основных модулей:

- **Telemetry** - модуль сбора и обработки данных с датчиков и устройств умного дома
- **Infra** - инфраструктурные компоненты, включая серверы конфигурации и обнаружения
- **Commerce** - модуль электронной коммерции, включающий подмодули:
  - Shopping Store - магазин товаров для умного дома
  - Shopping Cart - корзина покупок
  - Warehouse - управление складом
  - Interaction API - API для взаимодействия с другими модулями
  - Order - управление заказами
  - Payment - обработка платежей
  - Delivery - управление доставкой

## Технологический стек

Проект использует современный технологический стек:

- **Java 21** - основной язык программирования
- **Spring Boot 3.3.2** - фреймворк для разработки микросервисов
- **Spring Cloud 2023.0.3** - набор инструментов для разработки распределенных систем
- **Kafka 3.6.1** - для обработки потоковых данных
- **gRPC 1.63.0** - для высокопроизводительного удаленного вызова процедур
- **Avro 1.11.3** - для сериализации данных
- **Protobuf 3.23.4** - для сериализации структурированных данных
- **PostgreSQL** - реляционная база данных
- **Docker Compose** - для оркестрации контейнеров

## Модули проекта

### Telemetry

Модуль Telemetry отвечает за сбор, обработку и анализ данных с датчиков и устройств умного дома. Он включает в себя следующие подмодули:

- **Collector** - собирает данные с датчиков
- **Aggregator** - агрегирует данные
- **Analyzer** - анализирует данные
- **Serialization** - сериализует данные в различных форматах (Avro, Protobuf)

### Infra

Модуль Infra содержит инфраструктурные компоненты, такие как сервер конфигурации, сервер обнаружения сервисов и шлюз API.

### Commerce

Модуль Commerce реализует функциональность электронной коммерции для продажи товаров умного дома. Он включает в себя:

- **Shopping Store** - каталог товаров
- **Shopping Cart** - корзина покупок
- **Warehouse** - управление складом
- **Interaction API** - API для взаимодействия с другими модулями
- **Order** - управление заказами
- **Payment** - обработка платежей
- **Delivery** - управление доставкой

## Архитектура

Проект построен на микросервисной архитектуре с использованием современных подходов к разработке распределенных систем. Коммуникация между сервисами осуществляется через REST API, gRPC и сообщения Kafka.
